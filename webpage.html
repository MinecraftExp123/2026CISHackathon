How to Use Your Macros

1
Quick: Copy & Paste into Terminal
Click "Copy Command" on any macro below → open your terminal app (Terminal, iTerm, etc.) → paste with Ctrl+V (or Cmd+V on Mac) → press Enter. The command runs immediately.
2
Best: Import into the C Program (Keybindings)
This is the full MacroForge experience — run macros by pressing keybindings like ctrl+g.
a) Click the Export tab above → select macros → choose CSV format → click Download.
b) Open an external terminal (not VS Code terminal), navigate to the project folder, and run:
make && ./macro ~/Downloads/macroforge_export.csv
c) Choose [3] to run by name or [4] to run by pressing the keybinding you set.
Important: Use an external terminal (Terminal.app, iTerm) — VS Code's built-in terminal intercepts some shortcuts.
3
Alternative: Export as Shell Aliases
Go to Export tab → choose Shell format → Download the .sh file. In your terminal, run source ~/Downloads/macroforge_export.sh. Now just type the alias name (e.g. git_status) to run any macro. Aliases last until you close the terminal.
